登録した料理のレシピをgoogle検索できるように。

## 背景

PHPによるWEBサービスの勉強として作成。  
フレームワークは使わず。  


## 機能一覧

- ユーザー登録
- アカウント編集
- 退会
- ログイン
- ログアウト
- マイページ表示
- ご飯登録
- ご飯編集
- 登録したご飯一覧表示
- ご飯の詳細
- 料理のレシピのgoogle検索
- お気に入り登録、削除 (Ajax)
